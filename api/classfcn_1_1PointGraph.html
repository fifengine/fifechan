Detailed Description

Draws a scatter graph of individual points.

The graph is drawn by connecting the points with lines. The thickness of the lines can be set with setThickness. The graph can be set to be opaque or not with setOpaque. If the graph is opaque, it will be drawn with a solid color, otherwise it will be drawn with a transparent color.
